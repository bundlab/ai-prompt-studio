AI Prompt Studio is a full-stack application for managing AI prompts, built with FastAPI backend and React (Vite) frontend. It includes user management, database integration with PostgreSQL, and CI/CD pipelines.

## Screenshots

![Dashboard](https://via.placeholder.com/800x400?text=Dashboard+Screenshot)  
![User Profile](https://via.placeholder.com/800x400?text=User+Profile+Screenshot)

## Features

- User authentication and management
- RESTful API with FastAPI
- Responsive UI with React and TanStack Query for data fetching
- Dockerized for easy deployment
- CI/CD with GitHub Actions
